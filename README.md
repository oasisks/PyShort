# PyShort

A comprehensive tool that aims to use LLMs to determine market sentiment and potential movement base on historical data
and news. Leverage current leads in LLMs we aim to pair them with traditional neural nets to gain explainability and
trends within the stock market.

You can view a demo of this by downloading the required packages and then using streamlit to view the capabilities.
However, this does require an OpenAI Key. just use the env.example and fill in the OpenAI Key and you will be good to
go.
